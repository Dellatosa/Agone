# Mentions légales

Agone est un jeu de rôle français de **Mathieu Gaborit** et **Stéphane Marsan**, édité par **Multisim** en 1999. Il est basé sur la trilogie des *Chroniques des Crépusculaires*, les romans de **Mathieu Gaborit**.
